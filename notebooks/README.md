# AI Exercise Notebooks

These notebooks are starter workbooks, not finished solutions. Run the cells,
complete the TODO tasks, write short reflections, and commit your experiments.

| Notebook | Best time to start | What you will practice |
| --- | --- | --- |
| [01: Python and NumPy](01-python-and-numpy-practice.ipynb) | Days 003-004 | Arrays, indexing, summary statistics, and reflection |
| [02: pandas data story](02-pandas-data-story.ipynb) | Days 005-009 | DataFrames, grouping, charts, and communicating findings |
| [03: First classification model](03-first-classification-model.ipynb) | Days 021-030 | Splitting data, training, metrics, and error analysis |
| [04: Clustering and PCA](04-clustering-and-pca.ipynb) | Days 031-040 | Scaling, clustering, dimensionality reduction, and comparison |
| [05: Semantic search](05-semantic-search-concepts.ipynb) | Days 051-060 | TF-IDF vectors, similarity, retrieval, and evaluation |

## How to Run Them

Follow [the setup guide](../GETTING_STARTED.md), launch `jupyter notebook`, and
open a notebook from this folder. Use `Shift+Enter` to run the selected cell.
