# Project 05: Responsible AI Capstone

## Brief

Build a small AI application that is useful, measurable, and honest about its
limitations.

## Deliverables

- [ ] Define the user problem and a measurable outcome.
- [ ] Build a simple baseline before increasing complexity.
- [ ] Create an evaluation set and record results.
- [ ] Inspect failures and improve one important weakness.
- [ ] Identify privacy, bias, misuse, and reliability risks.
- [ ] Add human-review or escalation points where needed.
- [ ] Package the application so someone else can run it.
- [ ] Add setup, architecture, model-card, and limitations documentation.
- [ ] Prepare a five-minute demo and 100-day retrospective.

## Final Reflection

Explain what you would improve with more time, data, and user feedback.
