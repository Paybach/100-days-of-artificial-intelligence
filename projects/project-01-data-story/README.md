# Project 01: Data Story

## Brief

Choose a small public dataset and explain something meaningful with it.

## Deliverables

- [ ] Describe the dataset and the question you want to answer.
- [ ] Load the data into a notebook.
- [ ] Clean missing values, duplicates, and inconsistent types.
- [ ] Create at least three useful charts.
- [ ] Write five findings in plain English.
- [ ] Note two limitations or unanswered questions.
- [ ] Add a short README explaining how to run the notebook.

## Good Starter Ideas

Use weather, sports, transport, personal learning, or another topic that keeps
you curious enough to finish.
