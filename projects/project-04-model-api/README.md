# Project 04: Model API

## Brief

Serve one of your trained models behind a small API.

## Deliverables

- [ ] Choose a model and define its input and output.
- [ ] Create a prediction endpoint with FastAPI.
- [ ] Validate incoming data.
- [ ] Add a health endpoint.
- [ ] Add at least three automated API tests.
- [ ] Package the service with Docker.
- [ ] Document latency, failure cases, and expected usage.
- [ ] Explain how you would monitor the service.

## Stretch Task

Add a GitHub Actions workflow that runs the tests automatically.
