# 100 Days of Artificial Intelligence

A practical, beginner-friendly AI challenge: learn one focused topic each day,
build something small, and keep a visible record of the work.

## How the challenge works

1. Spend 30-90 minutes on the day's topic.
2. Complete the suggested hands-on task in [ROADMAP.md](ROADMAP.md).
3. Record what you learned in a folder under `days/`.
4. Mark the day complete in [PROGRESS.md](PROGRESS.md).
5. Use [RESOURCES.md](RESOURCES.md) when you need a trusted starting point.
6. Commit the result so the repository becomes your portfolio over time.

Start with [Day 001](days/day-001-ai-map-and-goals/README.md). For later days,
create a log from the reusable template:

```powershell
.\scripts\start-day.ps1 -Day 2 -Title "Python environment and notebooks"
```

## Learning path

| Days | Focus |
| --- | --- |
| 001-020 | Python, data, and the mathematical foundations of ML |
| 021-040 | Classical machine learning and responsible evaluation |
| 041-060 | Deep learning, computer vision, NLP, and transformers |
| 061-080 | LLM applications and MLOps |
| 081-100 | Responsible AI and a final portfolio project |
