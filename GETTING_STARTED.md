# Getting Started

Welcome. You do not need to master AI in one sitting. Aim for one focused
session of 30-90 minutes and leave a small trail of evidence: notes, notebook
experiments, a chart, model results, or a short reflection.

## Start Day 001

1. Open [the roadmap](ROADMAP.md) and read the Day 001 row.
2. Open [your Day 001 log](days/day-001-ai-map-and-goals/README.md).
3. Complete the AI-map exercise in that file.
4. Replace the reflection placeholder with what you learned.
5. Use [the Git guide](GIT_GUIDE.md) to push your work to GitHub.

## Set Up Notebook Work

From PowerShell:

```powershell
cd "C:\Users\noaha\OneDrive\Documents\New project\100-days-of-artificial-intelligence"
py -m venv .venv
.\.venv\Scripts\Activate.ps1
py -m pip install -r requirements.txt
jupyter notebook
```

Your browser will open. Start with [Notebook 01](notebooks/01-python-and-numpy-practice.ipynb).
The notebook exercises support the roadmap; you do not need to finish every
notebook in a single day.

## Start a Later Day

Open the [daily worksheet index](days/README.md), select the current day, and
follow its research prompts and hands-on steps. Each worksheet tells you what
evidence to add before updating [PROGRESS.md](PROGRESS.md) and publishing your
changes.

## Build Larger Projects

The [projects](projects/README.md) are portfolio checkpoints. Work on each one
gradually as you complete the related phase.
